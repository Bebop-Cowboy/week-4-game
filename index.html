Gem Collectors!

Can you match the gem quota?


Instructions

You will be provided quota of how many gems you will need to collect.

There will be four gems below.

By selecting a gem your collection will increase.

Each gem will contain a random amount.

To win, meet the quota. Match the number of gems ordered.

You lose the game when your collection reaches beyond the quota.

Remember, each time the game starts, the game will change the values of each gem.

Happy mining~!
